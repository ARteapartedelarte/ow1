Administración de sistemas

Mantener las aplicaciones funcionando y nunca perder los datos.

En este área hay dos perfiles claramente diferentes:

  • Administración de aplicaciones en la nube
  • Administración de una plataforma de cloud computing

Administración de aplicaciones en la nube

  • Idéntico rol para nubes públicas o privadas
  • Requiere conocimiento general de los recursos de la nube
  • Manejo de APIs REST tanto las de AWS como las de OpenStack
  • Orquestación y aprovisionamiento de infraestructura
  • Gestión de Imágenes
  • Control del ciclo de vida de las instancias
  • Almacenamiento de objetos
  • Monitorización, automatización y gestión de la configuración

Administración de una plataforma de cloud computing

  • Salvo para los administradores de Amazon, Azure o Google, la opción más probable es OpenStack
  • No es un puesto para "juniors" ;)
  • Es preciso dominar las tecnologías subyacentes: virtualización, redes y almacenamiento
  • Exige conocimiento profundo de la plataforma que se utilice, en el caso de OpenStack de un software muy amplio que evoluciona muy rápidamente.
  • Monitorización, automatización y gestión de la configuración
