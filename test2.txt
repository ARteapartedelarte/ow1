¿Qué relación hay entre una imagen y una instancia?

- Una imagen es una copia de una instancia
- Una instancia es una snapshot de una imagen
- Una instancia es una máquina virtual creada a partir de un disco virtual que llamamos imagen *
- Una instancia se puede descargar como una imagen

El sabor de la instancia...

- Me indica el tipo de sistema operativo que vamos a tener en la instancia
- Me indica las características hardware que va a tener la instancia creada *
- Me indica la configuración de red que va a tener la instancia
- Me indica los puertos que tiene abierto la instancia

Cuando suspendemos una instancia...

- Guardando su estado en disco *
- Guardamos su estado en memoria RAM
- Borramos la instancia
- Detenmeos la instancia

El proceso de crear una nueva imagen a partir de una instancia se llama:

- Redimención
- Copia de seguridad
- Descarga
- Instantánea *

¿Qué parámetro no es necesario inidcar en la creación de una instancia?

- IP flotente *
- Red a la que nos conectamos
- Nombre de la insyancia
- Sabor

Una IP Flotente...

- Siempre es una ip pública
- Siempre es una ip privada
- Siempre está asignada a una instancia
- Dependiendo del tipod e cloud, puede ser una ip pública o privada *


Al crear un estructura de red, ¿dónde se indican los parámetros del servidor DHCP?

- En la red
- En la subred
- En el router
- En el servidor de metadatos

¿En qué circunstancia no se puede asociar una IP flotante a una instancia?

- Cuando la instancia tiene asignada un ip fija.
- Cuando la instancia está conectada a una red privada
- Cuando la instancia está conectada a una red privada que está conectada a otra red interna con un router *
- Cuando la instancia tiene dos interfaces de red

¿Qué comando utilizamos para la creación de una instancia?

- nova start
- nova boot *
- nova resume
- nova resize

¿Qué elementos posee un pool en una balanceador de carga?

- Servidores, subred, algoritmo de balanceo
- Monitores, servidores web e ip flotentes
- Monitores, miembros y vip *
- Protocolos de red, puertos y configuraciónd e red

Indica que hace el comando swift upload web1 web2

- Sube el objeto web2 al contenedor web1 *
- Sube el objeto web1 al contenedor web2
- Sube dos ficheros al cloud
- Crea dos servidores web en el cloud

Si tenemos una instancia creada sobre un volumen

- El disco se borra cuando la instancia se termina
- La instancia tiene dos discos asociados
- Siempre se puede desaociar el volumen de la instancia
- El disco raíz de la instancia se considera información persistente *

La instrucción neutron port-create red --fixed-ip ip_address=10.0.15.39

- Crea un puerto al que no está conectado ningún dispositivo *
- Crea un puerto al que se conecta la instancia con ip fija 10.0.15.39
- Crea un puerto al que se asocia una ip flotante
- Se crea un nuevo router conectado a la red
